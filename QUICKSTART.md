# QuoteFlow - Quick Start Guide

## Getting Started in 60 Seconds

### For Proposal Creators

1. **Go to Builder**
   - Click "Get Started" button on landing page
   - Or navigate to `/builder`

2. **Fill in Your Details**
   - Company name and email
   - Client name
   - Description of proposal

3. **Add Line Items**
   - Describe the service/product
   - Set quantity and rate
   - Total calculates automatically
   - Add more items with "Add Line Item" button

4. **Share with Client**
   - Click "Share" button
   - Link is copied to clipboard
   - Send via email, message, or QR code

5. **Export if Needed**
   - Click "PDF" to download
   - Click "Print" for printing
   - Click "QR" to show scannable code

## Common Tasks

### How to Change Pricing
```
1. Open the builder (from landing page or back button)
2. Modify any line item rates
3. Quantities and totals update automatically
4. Click "Share" again to get new link
5. Send updated link to client
```

### How to Add Tax
```
1. Scroll to FINANCE section
2. Enter Tax Rate (%) - e.g., "10" for 10%
3. Totals recalculate automatically
4. Share updated proposal
```

### How to Change Client Name
```
1. Scroll to CLIENT section
2. Update "Client name" field
3. Preview updates instantly
4. Share new link
```

### How to Print Proposal
```
1. Click "Print" button in toolbar
2. Choose printer or "Save as PDF"
3. Configure paper size/margins
4. Print or save
```

### How to Download as PDF
```
1. Click "PDF" button in toolbar
2. File downloads automatically
3. Named as "PROPOSAL-XXXXX.pdf"
4. Can email or archive
```

### How to Show QR Code
```
1. Click "QR" button in toolbar
2. Modal opens with QR code
3. Client scans with phone camera
4. Proposal opens in browser
5. Click "Download QR" to save image
```

### How to Use Dark Mode
```
1. Click "Dark" button in toolbar
2. Interface switches to dark colors
3. Setting saves automatically
4. Works on all pages
```

### How to Start Over
```
1. Click "Reset" button
2. Confirm in dialog
3. All fields cleared
4. New proposal ID generated
5. Draft deleted from storage
```

## Troubleshooting

### 🔴 Share button says "Failed"
**Solution**:
- Close and reopen browser
- Check internet connection
- Try different browser
- Manually copy URL from toolbar

### 🔴 QR code won't scan
**Solution**:
- Ensure good lighting
- Move closer to camera
- Try different QR scanner
- Use the URL link instead

### 🔴 Proposal looks wrong in PDF
**Solution**:
- Try reducing text length
- Check line item descriptions
- Print preview first
- Try different PDF viewer

### 🔴 Data disappeared
**Solution**:
- Check browser storage not cleared
- Look in browser history for backups
- Check localStorage in DevTools
- Try browser recovery options

### 🔴 Can't copy share link
**Solution**:
- Allow clipboard permissions in browser
- Try in different browser
- Use print to PDF as alternative
- Manually copy URL from address bar

## Privacy & Security

### Your Data
- ✅ Stored only in your browser
- ✅ Never uploaded to servers
- ✅ No tracking or analytics
- ✅ No cookies or tracking pixels
- ✅ Works completely offline
- ✅ All processing client-side

### Sharing Safety
- ✅ Each share link is unique
- ✅ Can't be reverse-engineered
- ✅ Works only with full URL
- ✅ No signature verification needed

- Use notes section for terms only
- Revoke old links if not used
- Don't share with untrusted parties

## Getting Help

### Common Questions

**Q: Can I edit after sharing?**
A: Yes! Go back to builder and create a new proposal, then send updated link.

**Q: Can I see if client opened it?**
A: Not currently. QuoteFlow doesn't track views.

**Q: Will my proposal expire?**
A: No, links never expire. They work as long as needed.

**Q: Can I send to multiple clients?**
A: Yes, generate new link each time. Each share is independent.

**Q: What if I close browser?**
A: Your draft is auto-saved. Return to builder anytime.

**Q: Can client modify proposal?**
A: No, shared proposals are read-only for clients.

**Q: Is there a limit to proposals?**
A: No limit! Create unlimited proposals.

**Q: Can I use my phone to create?**
A: Yes! Mobile responsive design works great.
